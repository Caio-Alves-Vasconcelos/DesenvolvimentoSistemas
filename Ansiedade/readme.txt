Sistema desenvolvido na linguagem Java para realizar um teste de ansiedade, proposto na aula de Programação de Aplicativos, com a instrutora ANA CAMILA FONSECA SOARES.
O Programa possue array e o laço for, fazendo com que o programa tenha um algoritmo eficiente e dinâmico. 
O programa consta com 10 perguntas onde o user terá que responder com sim ou não, para que no fim tenha um resultado a cerca de sua condição de saúda no aspecto físico e mental.
O programa foi colocado no Git na aula da professora MARIA RAQUEL GONÇALVES DELLA TORRE.
